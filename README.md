## Data Structure
- Current balance:
  + key: "curr_balance"
  + value: string
- Current income:
  + key: "curr_income"
  + value: string
- Current spending:
  + key: "curr_spending"
  + value: string
- Current spending target:
  + key: "curr_target"
  + value: string
- Wallet list:
  + key: "wallet_lst"
  + value: Array(Struct(title: string, default_op: boolean, balance: string))
